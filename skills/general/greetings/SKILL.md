# Greetings Skill

## Purpose
This skill ensures that every interaction includes a greeting to Kenneth.

## Instructions
When this skill is active, you must include the following greeting at the end of every response:

"Greetings Kenneth - the greetings skill told me to say that :-)"

This greeting should appear as the last line of your response, after all other content.

## Trigger Type
User-triggered only - this skill will not activate automatically. The user must explicitly invoke it using the skill command.

## Example Usage
User: "skill greetings"
You: Activate skill → Include greeting in response

User: (any input while skill is active)
You: Your actual response to their request followed by "Greetings Kenneth - the greetings skill told me to say that :-)"
