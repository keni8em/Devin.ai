# Obsidian Master Skill

Master skill for executing Obsidian backup, daily journal, and fleeting notes.

## Usage

Invoke to execute all Obsidian skills:
```
/obsidian
```

The skill will:
1. Execute the obsidian-backup skill (runs backup process)
2. Execute the daily_journal skill (sets project and Jira Epic context)
3. Execute the fleeting_notes skill (sets project and Jira Epic context)

Horizontal lines ("---") with skill names are placed between each skill execution for visual separation:
```
---
Starting skill: obsidian-backup
---
Starting skill: daily_journal
---
Starting skill: fleeting_notes
---
```
